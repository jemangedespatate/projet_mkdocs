# Interrogation de SNT - Géolocalisation (Sujet A)

### Exercice 1 : Fonctionnement et définitions
1. Définissez brièvement le terme **Géolocalisation**.
2. Expliquez le principe de la **trilatération** utilisé par les récepteurs GPS pour trouver leur position sur Terre.
3. quelle est la ligne permetant de diferencier le nord du sud. de meme pour l'est et l'ouest. par quelle ville passe-t-elle ?
4. combien de signaux/satelitte sont necessaire pour pouvoir localiser precisement une personne grace au gps

### Exercice 2 : Calcul de distance
Un signal radio met **200 µs** pour voyager depuis un satellite jusqu'à votre smartphone. Sachant que la vitesse du signal est d'environ **300 m/µs**, calculez la distance qui les sépare en detaillent le calcul. donner ensuite le resultat en km.

### Exercice 3 : Conversion de coordonnées
La conversion est indispensable pour passer des formats GPS aux formats lisibles sur des cartes.

1. Convertissez la coordonnée suivante du format décimal vers le format degrés-minutes-seconde (DMS) : **45,50°**
2. Convertissez la coordonnée suivante du format degrés-minutes-seconde (DMS) vers le format degres : **12° 15' 10"**

### Exercice 4 : Trame NMEA
Voici une trame NMEA captée par un smartphone :  
`$GPGGA,083000.000,4317.0000,N,00522.0000,E,1,05,2.0,150.0,M,,,,0000*0F`

1. À quelle heure (UTC) cette position a-t-elle été enregistrée ?
2. Quelle est la lattitude à laquelle se trouve le capteur ?
3. Quelle est la longitude à laquelle se trouve le capteur ?
4. convertissez les donner trouver pour la lattitude et la longitude au format degree

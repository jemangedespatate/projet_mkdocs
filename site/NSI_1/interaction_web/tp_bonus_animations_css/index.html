    🚀 TP Bonus : Donnez vie à votre site web (Animations CSS)

    🎯 Objectifs

    Vos sites web sont maintenant fonctionnels et bien structurés ! Mais ils pourraient être encore plus dynamiques et interactifs. Dans ce TP bonus, nous allons utiliser le CSS pour créer des petites animations sans écrire une seule ligne de JavaScript.

    Note : Vous appliquerez ces nouveaux effets directement sur le site web que vous avez réalisé lors du TP précédent.

    🦋 Partie 1 : Des survols en douceur (transition et :hover)

    La pseudo-classe :hover (qui se rajoute dans le CSS) permet de changer le style d'un élément lorsque la souris passe dessus. La propriété transition permet de faire ce changement "en douceur" (de manière fluide) plutôt que de manière brutale ou instantanée.

    1.1 Animer vos liens de navigation

    Actuellement, votre menu change de couleur (ou de soulignement) de façon très hachée. Rendons cela plus fluide.

    Cible : Les balises <a> de votre menu de navigation.

    /* On ajoute la transition sur l'état "normal" de l'élément */
a {
    color: #3498db;
    /* La couleur changera en 0.4 secondes de façon très douce */
    transition: color 0.4s ease; 
}

/* On définit ce qui change AU SURVOL */
a:hover {
    color: #e74c3c;
}

    À vous de jouer : Appliquez une transition fluide sur les couleurs (texte ou fond) des liens de votre menu principal. N'hésitez pas à adapter le temps de transition !

    1.2 Effet de Zoom sur vos images

    C'est un effet très populaire pour les galeries d'images : l'image s'agrandit légèrement quand on la survole. Pour cela on utilise la transformation d'échelle : transform: scale().

    img {
    /* Toute transformation appliquée prendra 0.3 secondes */
    transition: transform 0.3s ease; 
}

img:hover {
    /* Au survol, l'image passe à 110% de sa taille d'origine (1.1) */
    transform: scale(1.1); 
}

    (Si les grandes images dépassent sur d'autres éléments, observez comment vos marges réagissent !)

    À vous de jouer : 1. Trouvez des images sur votre site. 2. Ajoutez-leur cet effet de zoom. Testez sur votre navigateur en rechargeant la page (touche F5).

    🎴 Partie 2 : Effet "Carte interactive"

    Nous allons créer un effet de "carte qui se soulève vers l'écran", que l'on retrouve sur de nombreux sites professionnels pour présenter des articles, des joueurs, des recettes, etc.

    Pour cela nous allons combiner 2 propriétés : - transform: translateY() pour déplacer la carte vers le haut (sur l'axe Y vertical). - box-shadow pour créer une ombre portée.

    À vous de jouer : Appliquez ces classes sur un <div class="carte"> contenant par exemple un sous-titre (h2), une petite image et un paragraphe.

    .carte {
    background-color: #f8f9fa;
    padding: 20px;
    border-radius: 10px;

    /* Une petite ombre par défaut sous la carte */
    box-shadow: 0 4px 8px rgba(0,0,0,0.1); 

    /* Transition sur TOUTES les propriétés qui vont changer plus tard */
    transition: all 0.3s ease;
}

.carte:hover {
    /* La carte remonte de 10 pixels (vers le haut, donc négatif en CSS) */
    transform: translateY(-10px); 

    /* L'ombre devient beaucoup plus grande et plus diffuse : la carte "décolle" */
    box-shadow: 0 15px 25px rgba(0,0,0,0.2); 
}

    🔄 Partie 3 : Animations continues avec @keyframes

    Jusqu'à cette partie, l'animation se déclenchait uniquement via une action de l'utilisateur (le passage de la souris). Avec les @keyframes, on peut créer une animation qui s'exécute automatiquement et qui tourne en boucle !

    3.1 Faire "flotter" votre titre principal

    Imaginons que vous voulez que le titre principal de votre page (votre h1) "flotte" doucement de haut en bas, en continu.

    On commence par décrire l'animation indépendamment de notre élément. On lui donne un nom, ici "flottaison" :

    @keyframes flottaison {
    0% { transform: translateY(0px); }
    50% { transform: translateY(-15px); }    /* Monte de 15px à la moitié de l'animation */
    100% { transform: translateY(0px); }     /* Redescend pour finir au point de départ */
}

    Puis on l'applique sur l'élément souhaité (ici le h1), de cette façon :

    h1 {
    /* 
       Propriété animation : [nom_de_lanimation] [duree] [type_de_mouvement] [répétition]
    */
    animation: flottaison 3s ease-in-out infinite;
}

    À vous de jouer : Appliquez cette animation sur un de vos titres, ou sur l'image centrale de votre page d'accueil pour lui donner un côté "magique".

    3.2 Le bouton "Pulse" (Battement de coeur)

    C'est une variante de l'animation précédente. On agit cette fois sur la taille (scale) plutôt que sur la position (translate). C'est extrêmement pratique pour attirer l'attention de l'utilisateur sur un élément important (comme un badge "NOUVEAU", une alerte ou un bouton de contact).

    @keyframes battement {
    0% { transform: scale(1); }
    50% { transform: scale(1.05); } /* L'élément grossit de 5% de sa taille */
    100% { transform: scale(1); }
}

.pulse {
    animation: battement 2s infinite;
}

    À vous de jouer : Créez un élément dans votre fichier .html (une icône, un bouton, un court paragraphe), associez-lui la classe class="pulse" et vérifiez que votre élément palpite !

    🏆 Défi final

    En utilisant les notions vues dans ce TP et une rapide recherche sur internet, essayez de créer une animation qui exécute une belle rotation sur un élément au passage de la souris !

    🚀 Partie 4 (Bonus+) : Un aperçu de la magie du JavaScript

    Nous avons vu que le CSS est fantastique pour des animations récurrentes ou gérées par le navigateur (comme le survol de la souris). Mais que faire si l'on veut réagir dynamiquement aux actions complexes de l'utilisateur ? C'est là que le JavaScript intervient !

    Avec JavaScript, on peut réagir aux clics précis, au défilement de la page, ou même suivre les mouvements de la souris en temps réel. Voici un petit exemple pour vous montrer sa puissance : créer un "fantôme" ou un "halo" qui suit le pointeur de votre souris !

    Tester la magie de JS

    Pour tester cette démonstration :

    1. Ajoutez ceci tout en bas de votre fichier HTML (juste avant la balise fermante </body>) :

    <!-- Notre élément qui va suivre la souris -->
<div id="curseur-suiveur"></div>

<!-- Le script qui va donner vie à notre élément -->
<script>
    // 1. On indique au JavaScript quel élément il doit manipuler
    const curseur = document.getElementById("curseur-suiveur");

    // 2. On demande au navigateur de surveiller (écouter) le mouvement de la souris
    document.addEventListener("mousemove", function(evenement) {
        // 3. À chaque micro-mouvement, on calcule la position X et Y
        // et on déplace dynamiquement la "bulle" avec transform!
        curseur.style.transform = `translate(${evenement.clientX}px, ${evenement.clientY}px)`;
    });
</script>

    2. Ajoutez ensuite ce style dans votre fichier .css pour donner une forme à votre curseur :

    #curseur-suiveur {
    width: 30px;
    height: 30px;
    background-color: #f39c12; /* Une belle couleur orange */
    border-radius: 50%;        /* Pour faire un rond parfait */

    position: fixed;           /* Pour qu'il vole au-dessus du reste de la page */
    top: -15px;                /* Pour que votre vraie souris soit au centre du rond */
    left: -15px;
    z-index: 9999;             /* Pour être sûr qu'il passe au-dessus de tout le reste */

    pointer-events: none;      /* CRÉCIAL : pour ne pas empêcher de cliquer sur vos vrais liens ! */
    transition: transform 0.1s ease-out; /* Un petit lissage du mouvement ("effet d'inertie") */
}

    Que s'est-il passé ? Sauvegardez vos fichiers et rechargez votre page web. Bougez votre souris... le rond orange vous suit !

    C'est là toute la différence : le CSS s'occupe de la présentation (la couleur, la forme), tandis que le JavaScript s'occupe de la logique et de l'interactivité (calculer des positions infinies en direct). C'est avec ce langage que l'on code les jeux par navigateur ou les réseaux sociaux modernes !

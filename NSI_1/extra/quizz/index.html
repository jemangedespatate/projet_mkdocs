    Grand Quiz NSI

    Phase 1 — Création des questions

    À vous de jouer : construisez les questions du quiz !

    ⏱️ Durée : 2 heures  |  NSI Première — 2025-2026

    🎯 Objectif de cette séance

    Vous allez **créer les questions** qui serviront lors du quiz de révision de la séance suivante. Chaque question que vous rédigez aujourd'hui sera posée à toute la classe lors du quiz.
    ⚠️ **Soyez rigoureux !** Vos questions seront lues et évaluées devant tous vos camarades. Une question mal formulée ou une réponse incorrecte sera signalée.

    📋 Les règles

    1
    Chaque élève doit rédiger au minimum 3 questions par thème.
    2
    Chaque question doit respecter un niveau de difficulté précis : facile, moyen ou difficile.
    3
    Chaque question doit être complétée sur une fiche individuelle selon le modèle fourni.
    4
    À la fin de la séance, toutes les fiches sont remises au professeur.

    🎯 Les niveaux de difficulté

    ### 🟢 Facile - Question de définition - Question vrai / faux - Identification simple d'un élément *Exemple : « Qu'est-ce qu'une variable ? »*
    ### 🟡 Moyen - Application d'une notion - Lecture ou complétion de code - Explication d'un mécanisme *Exemple : « Que renvoie `[1,2,3][1:3]` ? »*
    ### 🔴 Difficile - Raisonnement algorithmique - Analyse ou débogage - Question de complexité ou de comparaison *Exemple : « Combien de comparaisons pour une dichotomie sur 1024 éléments ? »*

    Thème 1

    Le Langage Python

    🐍 Thème 1 — Le Langage Python

    Notions à couvrir

    **Types de base** - `int`, `float`, `str`, `bool` - Conversions de types (`int()`, `str()`…) - Opérations arithmétiques et logiques **Variables & affectation** - Nommage, portée - Affectation simple et multiple
    **Structures de données** - Listes : `append`, `pop`, indexation, slicing - Tuples : immuabilité - Dictionnaires : accès par clé **Traitement de données en table** - Lecture de fichier CSV - Filtrage et tri de tables
    🖥️

    Thème 2

    La Machine

    🖥️ Thème 2 — La Machine

    Notions à couvrir

    **Modèle de Von Neumann** - Les 4 composants : UC, mémoire, entrées/sorties, bus - Cycle d'exécution : Fetch → Decode → Execute - Registres : PC, IR, ACC
    **Langage Assembleur** - Instructions : `MOV`, `LDR`, `STR`, `ADD`, `SUB`, `CMP`, `B`, `HALT` - Notion de registre - Lecture et interprétation de code assembleur
    **Système d'exploitation** - Rôle du noyau, gestion des processus - Notion de fichier, droits d'accès
    🔢

    Thème 3

    Les Algorithmes

    🔢 Thème 3 — Les Algorithmes

    Notions à couvrir

    **Algorithmes de tri** - Tri par insertion - Tri par sélection - Complexité : O(n²)
    **K Plus Proches Voisins (KNN)** - Principe de classification - Calcul de distance euclidienne - Influence du choix de k
    **Recherche dichotomique** - Prérequis : tableau trié - Principe de division par 2 - Complexité : O(log n)
    🌐

    Thème 4

    Web & Réseaux

    🌐 Thème 4 — Web & Réseaux

    Notions à couvrir

    **HTML** - Structure de base : ``, ``, `` - Balises courantes : titres, paragraphes, listes, liens, images - Formulaires **CSS** - Sélecteurs, propriétés courantes - Box model (margin, padding, border)
    **JavaScript** - Variables, conditions, boucles - Manipulation du DOM - Gestion d'événements **Réseaux** - Protocoles HTTP / HTTPS - Notion d'adresse IP et de DNS - Modèle client / serveur
    💡

    Thème 5

    Informatique Générale

    💡 Thème 5 — Informatique Générale

    Notions à couvrir

    **Intelligence Artificielle** - Principe du Machine Learning - Données d'entraînement - Biais algorithmiques **Histoire de l'informatique** - Figures clés : Turing, Von Neumann, Ada Lovelace - Grandes étapes de l'évolution des ordinateurs
    **Chiffrement & Sécurité** - Chiffrement symétrique et asymétrique - Hachage, mots de passe - Phishing, ingénierie sociale **Bonnes pratiques** - Licences logicielles (libre, propriétaire) - Données personnelles et RGPD - Impact environnemental du numérique

    ✅ Avant de rendre vos fiches

    ## Vérifiez pour chaque fiche :
    1
    Votre nom est bien indiqué.
    2
    Le thème est correctement renseigné.
    3
    Le niveau de difficulté est précisé.
    4
    La réponse correcte est clairement indiquée.
    5
    La question est compréhensible par quelqu'un qui ne l'a pas rédigée.
